# CNN Discord Bot and API
Discord bot and API made for CNN (Crusalis News Network).

# Repo structure
```
src/    
 ├─ commands/      - discord commands
 ├─ utils/         - util functions
 ├─ main.py
```

# Requirements/Installation
## Requirements:
- python
- pip

## Installation
1. Download/install python and pip: <https://www.python.org/downloads/>

2. Install pip packages:
```
pip install -r requirements.txt
```

# Run
1. Run `python src/main.py` in this repo root to start the bot.
